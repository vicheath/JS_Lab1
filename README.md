# JavaScript Lab 1 - Getting Started with JavaScript

This lab covers setting up a JavaScript development environment and writing a basic script using `console.log()` to output messages to the browser console. The script includes a standard greeting, a personalized message, and a fun fact.

## How to Run
1. Clone or download the repo
2. Open `index.html` in your browser
3. Press `F12` and go to the Console tab to see the output
